So I set out to build a GUI...

But which toolkit?

Tk?

  • Seriously? Dammit, I want something funky and modern!

WxWidgets?

  • Seems a bit odd having a x-platform library built on a x-platform library...
  • Plenty of forum posts indicating flakiness
  • Doesn't seem to be included in any of the matplotlib Windows distributions

GUI #1: GTK3

  • Wonderful toolkit and API (PyGObjects)
  • Horribly buggy designer (Glade)
  • Still, after a bit of effort, got something working ...
  • ... only to discover the matplotlib bundled with Ubuntu doesn't support GTK3
  • ... and it's not packaged for Windows yet either (well, not nicely anyway)
